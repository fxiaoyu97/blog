# Windows中杀死占用某个端口的进程

  1. 根据端口号查找对应的进程号
// 列出进程极其占用的端口，且包含 80
netstat -ano | findstr 8081
// 显示如下所示，pid进程号为15888
C:\Users\calos>netstat -ano | findstr 8081
  TCP    0.0.0.0:8081           0.0.0.0:0              LISTENING       15888
  TCP    [::]:8081              [::]:0                 LISTENING       15888
